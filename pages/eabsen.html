E-ABSEN PROVINSI KALSEL

Intro

Aplikasi ini dibuat bertujuan untuk mendisiplinkan seluruh ASN di Provinsi Kalsel sebanyak >15000 secara realtime. Yang mana aplikasi ini di bertujuan untuk mendisiplinkan kehadiran ASN, aplikasi ini mampu diterapkan disemua profesi seperti dokter, perawat, satpam, guru, dll.

Tech Stack

Dashboard Admin

Dashboard admin untuk check status kehadiran seluruh pegawai, fitur chat untuk menjawab pertanyaan oleh user disetiap instansi.

Data kehadiran

Data absensi karyawan bersifat realtime menggunakan socket.io

Data status raspberry

Halaman ini berfungsi untuk monitoring raspberry disetiap absensi.

Manajemen jadwal

Untuk menambahkan jadwal absensi, hal ini bisa dilakukan oleh admin kepada instansi yg meminta bantuan, atau oleh admin masing-masing instansi

Aktivitas error Raspberry

Untuk monitoring aktivitas error oleh raspberry.

Keterangan Absen

Untuk mengubah status kehadiran pegawai menjadi sakit atau cuti dan lain-lain. Cara penggunaannya adalah label status kehadiran di tarik ke kalendar pada tanggal tertentu
