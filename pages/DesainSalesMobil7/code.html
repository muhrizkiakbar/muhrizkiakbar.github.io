Paling Sering Ditanyakan

Kenapa harus lewat saya? expand_more
Saya bukan sekadar sales dealer. Saya adalah konsultan Anda yang akan membandingkan promo terbaik dari berbagai leasing dan memastikan Anda mendapatkan unit paling cepat dengan harga paling transparan. Saya dampingi dari awal cek stok sampai plat nomor jadi.
Apakah data dibantu sampai approve? expand_more
Tentu. Saya memiliki jaringan luas dengan berbagai leasing partner. Apapun kendala data Anda (BI Checking, domisili, dsb), komunikasikan saja dengan saya. Saya akan carikan solusi terbaik agar pengajuan Anda tetap disetujui.
Apakah melayani tukar tambah? expand_more
Iya, saya melayani trade-in (tukar tambah) mobil lama Anda merk apapun dengan unit baru. Saya bantu pastikan mobil lama Anda dihargai paling tinggi di pasaran.
